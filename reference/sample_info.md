# Sample info for the Piezo1 Xenium data

This is a data frame with two columns: sample and condition. Condition
is the experimental condition. PI14 stands for 14 days post complete
injection, and PI7 is 7 days. Injection means saline was injected into
an implant beneath the skin to expand the skin.

## Usage

``` r
sample_info
```

## Format

A data frame
