
# Wayfarer

<!-- badges: start -->
<!-- badges: end -->

This is the package presented in the [preprint](https://www.biorxiv.org/content/10.64898/2026.02.16.706245v1.abstract) Wayfarer: A multiscale framework for spatial analysis of tumor progression

## Installation

You can install the development version of Wayfarer from [GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("Computational-Morphogenomics-Group/wayfarer")
```

This package will be submitted to Bioconductor soon.
